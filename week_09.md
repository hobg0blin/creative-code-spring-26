### Activity: Vibecoding vs. Handcoding

If you don't use AI, let me know and you can handcode every part of this.

Group A (John Henry): handcode
Group B (Steam engine): vibecode

Prompts (pick one):
- "In p5, make a creature that responds to mouse movement"
- "In p5, make a generative pattern inspired by weather"
- "In p5, create a sketch where shapes grow and change over time"

**Round 1:**
- Group A: write it from scratch in p5 using what you've learned so far. You can use google and previous materials from class but no chatbots. It is *fine* if you don't finish or your code doesn't work, this is an experiment.
- Group B: use ChatGPT, Claude, or whatever LLM you want. You can keep prompting it to change your results, but don't read the code it puts out.

**Round 2:**

- Group A: now, use the AI of your choice to make any changes or updates you *wanted* to make but didn't have time to. If you don't use AI at all, take this time to keep working on it by hand.
- Group B: now take the AI-generated code and make it *yours* - try and add a personal aesthetic touch to it **without using AI**. Change colors, add interactions, break it in interesting ways, make it weird.
